⚙
 Cara Menjalankan Aplikasi Lokal 
1. Pastikan PHP sudah terinstall di komputer. 
2. Buka terminal/Git Bash di dalam folder `src`. 
3. Jalankan perintah: `php -S localhost:8000` 
4. Buka browser dan akses: `http://localhost:8000`
